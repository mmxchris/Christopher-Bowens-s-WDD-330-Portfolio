Functinal Programming

A major part of is Functional Programming is the use of pure function. A pure function has three rules:

  • The return value only depends on the values provided.
  • It does not change values or data outside the function.
  • Referential transparency - when given the same arguments it will return the same results.

    • To follow these rules a function must have at least one argument and return a value.
